# SimuBlock

**SimuBlock** is a unified blockchain simulator designed for:

- Application Demos (token transfers, state changes)
- Performance Research (TPS, block propagation time)
- Security Research (selfish mining behavior)

It is lightweight, fully Python-based, and cloud-deployable on free platforms like Render.

## Features

- Multi-node blockchain network
- Proof-of-Work mining
- Simulated network latency
- Token-based application layer
- Basic performance metrics
- Built-in selfish miner model
- Cloud-ready experiment runner
